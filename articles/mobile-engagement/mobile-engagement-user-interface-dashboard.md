# <a name="how-to-view-and-manage-charts-of-data-about-your-application"></a>Hogyan tekintheti meg és kezelheti a alkalmazással kapcsolatos adatok diagramok
A felhasználói felület irányítópult szakasza lehetővé teszi a felhasználóknak, az idő maximalizálása, mert a pontos szükséges információt keres a analytics szakasz helyett testre szabott diagramok létrehozása. Irányítópult jeleníti meg az alkalmazás és az aktív felhasználót megszámolja, hogy az alkalmazás egy adott idő alatt a különböző verzióira vonatkozóan alakulása (óra vagy nap/hét/hónap/testre szabott). A trendek látható az utóbbi 7 napból alapulnak.

### <a name="see-also"></a>Lásd még:
-  [Hibaelhárítási útmutatójának - elemzés][Link 21]

## <a name="dashboard"></a>Irányítópult
- Egy üres dashboard diagramok hozzáadhatja a minden diagram Analytics szakaszában a felhasználói felület, amelyeket szeretne láthat egy irányítópult menü a "Küldés irányítópult" művelet kiválasztásával.
 
## <a name="-add-a-dashboard"></a>+ (Hozzáadás irányítópult)
- A "+" menüszalag menüpont használatával, amelyek bemutatják a menüszalag menü új irányítópult létrehozása.
 
## <a name="gear-menu"></a>Fogaskerék menü
- Egy irányítópult a menü használatával törlése, nevezze át, a megosztás vagy az irányítópult ütemezett (E-mail fogadása). A kimutatás lehetőséget is használhatja az ütemtervet, amelynél meg szeretné kapni az egyéni irányítópult mailben meghatározásához. Napi, heti vagy havi jogosulttá ütemezheti. Megadhatja, hogy a, és a napot a hét, vagy ha szeretné, hogy az e-mailt kap a hónap napja időtartamát. E-mail címét és időzóna alapján a beállította a saját fiók szakaszában a felhasználói felület beállításait.
