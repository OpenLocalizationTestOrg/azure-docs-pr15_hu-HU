# <a name="azure-toolkit-for-intellij"></a>Azure eszközkészlete IntelliJ

Az Azure eszközkészlete IntelliJ biztosít a sablonok és funkciókat, amelyek lehetővé teszik, hogy könnyen létrehozása, kialakítása és tesztelése és helyezhetnek üzembe a IntelliJ fejlesztői környezet segítségével Azure alkalmazásokat. Megnyitás projektben, amelynek forráskód részére elérhető a webhelyen a project a MIT-licenc:

* <https://github.com/Microsoft/Azure-Tools-for-Java>

> [AZURE.NOTE] Az Azure eszközkészlet az Holdas ide is van. További tudnivalókért lásd: [Azure eszközkészlete Holdas].
