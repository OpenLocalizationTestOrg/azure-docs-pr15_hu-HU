# <a name="upgrade-your-free-azure-subscription-to-pay-as-you-go"></a>Ingyenes Azure előfizetést kirovó frissítése

Az [Ingyenes próbaverziót](https://azure.microsoft.com/free/) vagy [Microsoft Imagine](https://azure.microsoft.com/offers/ms-azr-0144p/) előfizetés az Azure-fiók portálról [kirovó](https://azure.microsoft.com/offers/ms-azr-0003p/) frissíthet. 

1. Jelentkezzen be az [Azure-fiók portálra](https://account.windowsazure.com/subscriptions) , és nyissa meg az **előfizetések** lapon.

2. Az előfizetés állapotának csoportjában jelölje be a **Frissítés most** transzparens.

    ![Értesítést kapott az ingyenes próbaverziót kirovó frissítése](./media/billing-upgrade-azure-subscription/billpage.png)

3. Erősítse meg a frissítést.

    ![Az előfizetés-re történő frissítésének megerősítése](./media/billing-upgrade-azure-subscription/Upgrade.png)

Ingyenes próbaverzió kirovó frissítéskor bármely maradtak vissza az ingyenes próbaverziót, például a 30 napos korlátozása vagy $200 hitel előnyöket folytatni az egyik erejéig. Az előfizetés frissítés azt követően fordul elő.
