# <a name="manage-log-analytics-using-azure-resource-manager-templates"></a>Azure erőforrás-kezelő sablonok használata a napló Analytics kezelése

Használhatja a [Azure erőforrás-kezelő sablonok] (... / azure-resource-manager/resource-group-authoring-templates.md) létrehozása és konfigurálása a napló Analytics-munkaterületek. A feladatokat végezheti el a sablonok példája:

+ Munkaterület létrehozása
+ A megoldás hozzáadása
+ Mentett keresés létrehozása
+ Egy számítógép csoport létrehozása
+ A Windows agent az IIS naplók számítógépekről gyűjteménye engedélyezése
+ Számítógépeken Linux és a Windows teljesítmény számláló összegyűjtése
+ Syslog Linux számítógépeken eseményeinek összegyűjtése 
+ Események gyűjt a Windows-eseménynaplók
+ Egyéni események naplógyűjtés
+ A log analytics agent hozzáadása egy Azure virtuális gépen
+ Log analytics Azure diagnosztika eszközzel gyűjtött adatok indexeléséhez konfigurálása


Ebben a cikkben sablon minták, amelyek bemutatják a beállításokat, amelyek hajthatja végre sablonok közül.

## <a name="create-and-configure-a-log-analytics-workspace"></a>Létrehozása és konfigurálása a napló Analytics-munkaterület

A következő sablon példa azt mutatja be, hogy miként:

1.  Munkaterület létrehozása
2.  A munkaterület megoldások hozzáadása
3.  Mentett keresés létrehozása
4.  Egy számítógép csoport létrehozása
5.  A Windows agent az IIS naplók számítógépekről gyűjteménye engedélyezése
6.  Logikai lemez perf számláló összegyűjtése Linux számítógépekről (% használt Inodes; Ingyenes megabájt; % Használt terület; / Mp átvitelek; / Mp Olvasás; Lemezre írása/sec)
7.  Syslog események összegyűjtése Linux számítógépekről
8.  Hiba és figyelmeztetés események az alkalmazás eseménynaplójában Windows számítógépekről összegyűjtése
9.  A memória rendelkezésre álló memória teljesítmény számláló gyűjt a Windows rendszerű számítógépeken
10. Egyéni naplózási összegyűjtése 
11. IIS naplókról és a Windows-eseménynaplók tárterület-fiókhoz Azure diagnosztika által írt összegyűjtése

### <a name="deploying-the-sample-template"></a>Az űrlapsablonok telepítése

Az űrlapsablonok telepítése:

1. Mentse a csatolt minta egy fájlba, például`azuredeploy.json` 
2. Szeretné, hogy a kívánt konfiguráció a sablon szerkesztése
3. Használatával a PowerShell vagy a parancssorból a sablon

#### <a name="powershell"></a>A PowerShell

`New-AzureRmResourceGroupDeployment -Name <deployment-name> -ResourceGroupName <resource-group-name> -TemplateFile azuredeploy.json`

#### <a name="command-line"></a>Parancssor

```
azure config mode arm
azure group deployment create <my-resource-group> <my-deployment-name> --TemplateFile azuredeploy.json
```
