Webalkalmazást hoz létre, a Microsoft Azure biztosít rövid altartományt azurewebsites.net tartományban, a felhasználók hozzáférhet az egy URL-CÍMÉT, például a http:// használatával webalkalmazást&lt;SajátWebalk&gt;. azurewebsites.net. Egyéni tartománynevet (például contoso.com), a felhasználók további felismerhető tartománynév biztosítása érdekében a web app alkalmazással is társíthat.
